# Aplicacion Socket-Server

Reconstruir modulos de Node
```
npm install
```

Transpilar Typescript -> Javascript
```
tsc -w
```

Levantar Servidor Node
```
nodemon | node
```
Repositorio github del Frontend Angular [Angular](https://github.com/josseas/ng-socket)
